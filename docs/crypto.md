## Crypto()

### Syntax
```
Crypto()
```

### Parameters
<dl>
</dl>

### Examples
```
// encode>decode base32const encoded = Crypto.base32Encode("data");console.log(encoded); //MRQXIYI=const decoded = Crypto.base32Decode(encoded);console.log(decoded); //data
```
