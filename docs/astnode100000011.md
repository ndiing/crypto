# Crypto.encrypt
AES Encryption

## Syntax
```js
encrypt(data,options)
```

### Parameters
Name | Type | Default
--- | --- | ---
`data` | `String` | 
`options` | `Object` | 
`options.algorithm` | `String` | aes256
`options.key` | `String` | 
`options.iv` | `String` | 
`options.encoding` | `String` | hex

### Return value
`String`
